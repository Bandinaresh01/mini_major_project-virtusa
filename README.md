# Mini & Major Projects Showcase

Welcome to the **Mini & Major Projects Showcase** repository! This repository acts as a comprehensive technical portfolio demonstrating proficiency across **Python, Java, and SQL** using both fundamental problem-solving concepts and advanced architectural frameworks (such as Natural Language Processing, Web App deploying, and Object-Oriented design patterns).

This repository is split into two main sections:

---

## 📂 1. Mini Projects Showcase

A collection of 9 standalone, highly focused projects showcasing clean logical implementation, structured databases, and fundamental syntax efficiency.

- **`sql_digital_library`**: SQL schemas resolving auditing and inactivity filters.
- **`java_password_validator`**: Java loop parsing verifying strict Regex validation conditions.
- **`python_farecalc`**: Python optimization scripting utilizing dictionaries measuring time-based surge pricing algorithms.
- **`sql_retail_insights`**: RDBMS SQL analytic views deriving dead-stock metrics and categorized financial yields.
- **`java_finsafe`**: Robust Object-Oriented Exception handling logic tracking abstracted Transaction arrays.
- **`python_opsbot`**: Back-end CLI data utility parsing logs and extracting anomalies automatically into targeted subsets.
- **`sql_logistics_tracker`**: Extraneous SQL `JOIN` constraints ranking supplier delivery reliability.
- **`java_smartpay`**: Interface implementations modeling dynamic tiered-slab utility invoicing models.
- **`python_content_sanitizer`**: Advanced Python RegEx substitutions providing redaction capabilities and hyperlink extractions.

---

## 📂 2. Major Projects Showcase (`major_project/`)

An advanced level execution containing 9 more complex projects mimicking full-stack concepts, external library usage, UI rendering, and in-depth analytical schema mapping. 

- **`python_expense_tracker`**: A command-line financial suite charting dynamic data to visual outputs using `matplotlib`.
- **`sql_retail_sales_analysis`**: Robust BI SQL algorithms solving high-value enterprise queries including inactive churn and volume limits.
- **`java_library_management`**: Native Java Collections dynamically allocating resources mimicking active runtime DB transactions mapping local time-series differentials using `java.time`.
- **`python_resume_analyzer`**: An explicit Machine Learning/NLP heuristic script mathematically validating job candidate fit utilizing Set tokenization.
- **`sql_hospital_management`**: High integrity multidimensional relational RDBMS schema mapping abstract Patient -> Doctor -> Diagnostic networks.
- **`java_quiz_system`**: An interactive application that registers timing loops testing durations dynamically with on-the-fly Question generation.
- **`python_weather_dashboard`**: A full HTML/web app deployed over **Python Flask** mapping external dataset regression algorithms directly onto the DOM utilizing **Chart.js**.
- **`sql_movie_recommendation`**: Emulates streaming-service collaborative filtering algorithms solving predictive suggestion equations via SQL matrices.
- **`java_banking_system`**: Deep OOP demonstration involving Abstract Parent classes orchestrating polymorphic subclasses, Overload protections, and authentication loops.
